# book-spring-ai

이 저장소는 한빛미디어 도서 『이것이 Spring AI다』를 읽으며  
Spring AI 기반 AI 애플리케이션 개발을 실습한 코드를 정리한 공간입니다.

저자는 Spring Boot 환경에서 LLM을 활용한 다양한 AI 기능을 구현하는 방법을 설명하며,  
텍스트 대화, 프롬프트 엔지니어링, 멀티모달, RAG, Tool Calling, Agent까지  
**Spring 기반 생성형 AI 애플리케이션 개발 전 과정을 실습 중심으로 학습할 수 있습니다.**
